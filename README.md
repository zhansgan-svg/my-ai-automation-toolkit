# AI 多智能体自动化工作平台
一款面向个人开发者的智能体编排与自动化调试工具，专注为开发者提供低代码智能体搭建、任务自动化编排、代码生成调试、文档结构化处理能力，适用于个人开发、业务流程自动化、学习验证大模型能力。

## 产品流程
1. 智能体角色配置 — 自定义智能体角色、分工、权限、工作模式，包含编码代理、审查代理、文档代理、数据分析代理
2. LLM 参数调度 — 统一封装通用大模型接口，支持动态切换模型温度、上下文窗口、最大生成长度、流式输出参数
3. 自动化任务编排 — 基于链式工作流，实现多智能体自主对话、任务拆解、递进执行、自动纠错
4. 代码工程能力 — 自动生成可运行代码、单元测试、代码重构、异常捕获，支持本地沙箱运行验证
5. 日志统计与用量分析 — 全局记录 Token 消耗、请求次数、输入输出长度、模型耗时，自动生成月度用量报表


## 技术栈
后端：Python 3.11 + FastAPI + Autogen 0.2 + SQLAlchemy + SQLite + LLM OpenAI/Anthropic通用接口 + Token用量统计中间件
前端：Vue3 + Vite + TypeScript + Pinia（简易可视化控制台，用于调试智能体）
